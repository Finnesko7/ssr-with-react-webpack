# SSR (Server Side Rendering) with React

## Pre requirements
* [Node.js](https://nodejs.org/)

## Libraries docs
* [Express](http://expressjs.com/)
* [React](https://reactjs.org/)
* [Webpack](https://webpack.js.org/)
* [Babel](https://babeljs.io)


## Getting started
```bash
$ npm i
$ npm run dev
```
## Product
```bash
$ npm run start
```

Service available on `localhost:3000`
